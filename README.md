# Belle & Bloom - Assignment 4 E-Commerce Site

Belle & Bloom is a high-fidelity interactive e-commerce prototype created for SEG3125 Assignment 4. The store sells women's clothing and beauty products.

## Main features

- Functional faceted search by category, size, color, price, material/finish, and product type
- Product browsing and product detail modal
- Cart with quantity control
- Checkout process with visible steps: Cart, Information, Payment, Confirmation
- Payment form prototype, not connected to a real payment system
- Short post-shopping survey
- Contact section with Carla's email
- Internationalization/localization notes through a language toggle and locale-aware presentation
- Heuristic evaluation support through clearly visible feedback, help, error prevention, consistency, and recovery controls

## Deployment

Use the following Netlify settings:

- Branch: `main`
- Base directory: leave empty
- Build command: `npm run build`
- Publish directory: `dist`
